Azure App Registration — Required Permissions

  1. Go to Azure Portal → App registrations → New registration
  2. Name it Intermedic Support Desk, single-tenant
  3. Under Certificates & secrets → create a client secret, copy the value
  4. Under API permissions → Add → Microsoft Graph → Application permissions:
    • Mail.ReadWrite — read & mark emails as read
    • Mail.Send — send outbound notifications
  5. Click Grant admin consent
  6. Copy Application (client) ID and Directory (tenant) ID
  7. Add to .env on the server: 
    AZURE_TENANT_ID=your-tenant-id
AZURE_CLIENT_ID=your-client-id
AZURE_CLIENT_SECRET=your-client-secret
O365_MAILBOX=informatics@intermedic.com
  8. Restart the service: sudo systemctl restart intermedic-desk
